# 부키부키(BookyBooky)
## 개요

**`프로젝트 이름`** 부키부키(BookyBooky)

**`프로젝트 설명`** 부키부키(BookyBooky)는 사용자의 올바른 독서 습관 형성과 목표 달성을 도와주는 앱입니다.

**`프로젝트 기간`** 2023. 03. 13 ~ 2023. 08. 31 (예정)
